### 安装环境
项目运行基于`Nodejs`+`Mysql`，你需要先安装必要的环境，不会安装 自行Google

推荐使用宝塔`Nodejs`管理器 + `Supervisor`进程守护插件

`Nodejs`官网：https://nodejs.org/


### 安装依赖

我习惯使用 `npm`

> npm install

### 导入数据库

导入 `mi_sd.sql` 推荐版本 `Mysql5.7`

### 运行

+ 直接运行
    > node ./app.js
+ 后台运行
    > nohup node ./app.js &

**推荐使用宝塔的`Supervisor`进程守护插件进行运行**
